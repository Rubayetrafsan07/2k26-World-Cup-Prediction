# 🌍 World Cup 2026 Prediction Engine

An end-to-end **data science and machine learning project** that simulates and predicts FIFA World Cup 2026 outcomes using historical football data, player statistics, and probabilistic modeling.

---

## ⚽ Project Overview

This project explores football analytics by building a complete pipeline that goes from raw data to **match prediction, player clustering, and full tournament simulation**.

The system combines:
- Machine Learning models for match outcome prediction
- Player-level analytics and clustering
- Team strength modeling
- Monte Carlo simulation for tournament outcomes
- Interactive Streamlit dashboard

---

## 🚀 Key Features

### 🧠 Machine Learning Models
- XGBoost / Random Forest model for match prediction
- Predicts **Win / Draw / Loss probabilities**
- Trained on historical international match data

### 👥 Player Analytics
- Engineered performance metrics:
  - Goals per 90
  - Assists per 90
  - Cards per 90
  - Value efficiency
- Player clustering using **KMeans**
  - Elite Attackers
  - Creative Players
  - Defensive Players
  - Aggressive Players
  - World-Class Players

### 🏆 Team Strength Modeling
- Built team-level ratings from player data:
  - Attack strength
  - Midfield strength
  - Defense strength
  - Goalkeeper strength
  - Squad quality index

### 🌍 World Cup Simulation
- Full tournament simulation:
  - Group stage
  - Knockout rounds
  - Final winner prediction
- Monte Carlo simulation for probability estimation

### 📊 Interactive Dashboard
Built with **Streamlit**, featuring:
- Run World Cup simulation
- Display predicted winner
- Show Monte Carlo probabilities
- Interactive UI for exploration
